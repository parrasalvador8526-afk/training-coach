1. Introducción

Objetivo: Proporcionar herramientas de análisis avanzado que complementen las 10 metodologías de NEXUS, permitiendo tomar decisiones informadas sobre entrenamiento, nutrición y periodización.

2. Herramienta 1: Potencial Genético

Calcula el límite máximo de masa muscular que puedes alcanzar de forma natural usando fórmulas científicas.

Fórmulas Utilizadas:

Fórmula Autor Datos Requeridos Precisión
FFMI Máximo Martin Berkhan Solo altura Estimación rápida
Detallada Casey Butt Altura + muñeca + tobillo Muy precisa

Uso en Consola:

// Fórmula Berkhan (altura en cm) HerramientasNEXUS.PotencialGenetico.calcularBerkhan(175) // Resultado: 76.6 kg de masa magra máxima // Fórmula Casey Butt (altura, muñeca, tobillo en cm) HerramientasNEXUS.PotencialGenetico.calcularCaseyButt(175, 17, 23) // Resultado: { pesoCompeticion: X, peso10Grasa: Y, peso15Grasa: Z }

3. Herramienta 2: Readiness Score

Evalúa tu nivel de fatiga acumulada y determina si necesitas un deload.

Factores Evaluados:

  • Metodología actual (cada una tiene diferente impacto en SNC)
  • Semana del ciclo (fatiga se acumula)
  • Horas de sueño (crítico para recuperación)
  • Nivel de estrés (1-10)
  • Días desde último deload

Interpretación del Score:

Score Nivel Acción
80-100 🟢 ÓPTIMO Entrena con máxima intensidad
60-79 🔵 BUENO Entrena normal, monitorea
40-59 🟡 MODERADO Reduce volumen 20-30%
20-39 🟠 BAJO Considera deload
0-19 🔴 CRÍTICO ⚠️ DELOAD OBLIGATORIO

Uso en Consola:

HerramientasNEXUS.ReadinessScore.calcularScore({ metodologia: 'gvt', // ID de metodología semanaActual: 3, // Semana del ciclo horasSueno: 7, // Horas promedio nivelEstres: 5, // 1-10 diasDesdeDeload: 25 // Días }) // Resultado: 59 (MODERADO)

4. Herramienta 3: Sincronización Circadiana

Optimiza tus horarios de entrenamiento según tu cronotipo natural.

Cronotipos:

Cronotipo Hora Óptima Entreno Hora Pico Fuerza Carbos Concentrar
Alondra (madrugador) 6:00-10:00 9:00-11:00 6:00-14:00
Neutral 10:00-18:00 14:00-17:00 8:00-18:00
Búho (nocturno) 16:00-21:00 17:00-20:00 14:00-22:00

Uso en Consola:

// Determinar cronotipo (hora despertar, hora dormir en formato 24h) HerramientasNEXUS.SincronizacionCircadiana.determinarCronotipo(7, 23) // Resultado: 'buho' // Obtener recomendaciones HerramientasNEXUS.SincronizacionCircadiana.obtenerRecomendacion('buho') // Resultado: { entreno: '16:00-21:00', fuerza: '17:00-20:00', carbos: '14:00-22:00' }

5. Herramienta 4: Tiempo hasta Objetivo

Estima semanas/meses realistas para alcanzar tus metas de composición corporal.

Tasas Realistas:

Nivel Pérdida Grasa/Semana Ganancia Músculo/Mes
Principiante 0.8% peso corporal 0.9-1.8 kg
Intermedio 0.6% peso corporal 0.45-0.9 kg
Avanzado 0.4% peso corporal 0.22-0.45 kg

Uso en Consola:

// Tiempo para perder grasa (peso, %grasa actual, %grasa objetivo, nivel) HerramientasNEXUS.TiempoHastaObjetivo.semanasParaPerderGrasa(80, 20, 12, 'intermedio') // Resultado: { pesoObjetivo: 72.7, pesoAPerder: 7.3, semanas: 15, meses: 3.5 } // Tiempo para ganar músculo HerramientasNEXUS.TiempoHastaObjetivo.mesesParaGanarMusculo(65, 75, 'intermedio') // Resultado: { musculoAGanar: 10, meses: 40, anos: 3.3 }

6. Herramienta 5: Matriz de Transición

Guía de cómo cambiar entre metodologías sin perder adaptaciones.

Compatibilidad entre Metodologías:

⚠️ Regla General: Metodologías del mismo tipo (TM-TM o EM-EM) tienen mayor compatibilidad. Transiciones entre tipos opuestos (Heavy Duty → GVT) requieren más tiempo.
Transición Compatibilidad Semanas Transición
Heavy Duty → Blood & Guts 9/10 1 semana
GVT → SST 9/10 1 semana
Y3T → DUP 8/10 1 semana
Heavy Duty → GVT 3/10 4 semanas

Uso en Consola:

HerramientasNEXUS.MatrizTransicion.obtenerRecomendacion('heavy-duty', 'gvt') // Resultado: { // semanas: 4, // compatibilidad: 3, // recomendacion: "Metodologías muy diferentes. Incluir semana de deload entre ambas." // }

7. Herramienta 6: Grupos Rezagados

Identifica qué músculos necesitan más atención basándose en proporciones ideales.

Proporciones Ideales (Culturismo):

Proporción Ratio Ideal Ejemplo
Hombros / Cintura 1.618 (Proporción Áurea) 115cm / 71cm
Pecho / Cintura 1.4 100cm / 71cm
Bíceps / Antebrazo 1.2 40cm / 33cm
Muslo / Pantorrilla 1.3 60cm / 46cm

Uso en Consola:

// Calcular scores de proporción const scores = HerramientasNEXUS.GruposRezagados.calcularScore({ hombros: 110, pecho: 100, cintura: 80, biceps: 35, antebrazo: 30, muslo: 55, pantorrilla: 40 }); // Identificar rezagados HerramientasNEXUS.GruposRezagados.identificarRezagados(scores) // Resultado: [{ grupo: 'hombros', score: 85, prioridad: 'ALTA', volumenExtra: '+50%' }]

8. Guía de Uso en Consola

✅ Pasos para usar las herramientas:
  1. Abre la aplicación NEXUS en Chrome
  2. Presiona F12 para abrir DevTools
  3. Ve a la pestaña "Console"
  4. Escribe HerramientasNEXUS y presiona Enter para ver todas las herramientas
  5. Usa las funciones como se muestra en los ejemplos

Verificar que todo funciona:

// Test rápido - copia y pega esto en consola: console.log('HerramientasNEXUS:', typeof window.HerramientasNEXUS); console.log('Potencial 175cm:', HerramientasNEXUS.PotencialGenetico.calcularBerkhan(175)); console.log('Readiness:', HerramientasNEXUS.ReadinessScore.calcularScore({ metodologia:'y3t', semanaActual:2, horasSueno:8, nivelEstres:3, diasDesdeDeload:14 }));
